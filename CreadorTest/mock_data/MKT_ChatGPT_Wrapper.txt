MKT_ChatGPT_Constants

-----
La clase MKT_ChatGPT_Wrapper es una clase pública que contiene dos subclases: BodyToChat y ResponseChat.

La subclase BodyToChat contiene las propiedades model, prompt, temperature y max_tokens. Tiene un constructor que toma un parámetro prompt y establece valores iniciales para otras propiedades usando constantes de la clase MKT_ChatGPT_Constants. También tiene un método getBody() que serializa la instancia en un string JSON.

La subclase ResponseChat contiene una lista de objetos Text llamada choices. La clase Text tiene una única propiedad de tipo string llamada text.
