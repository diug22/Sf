MKT_ChatGPT_Constants
-----
La clase MKT_ChatGPT_Wrapper contiene dos clases internas: BodyToChat y ResponseChat. La clase BodyToChat se utiliza para construir el cuerpo de la solicitud de chat y tiene un constructor que acepta un parámetro 'prompt'. También utiliza constantes de la clase MKT_ChatGPT_Constants para establecer valores predeterminados para 'model', 'temperature' y 'max_tokens'. La función getBody() devuelve la representación JSON del objeto BodyToChat. La clase ResponseChat se utiliza para manejar la respuesta del chat y contiene una lista de objetos Text, donde cada objeto Text tiene una propiedad 'text'.
